Versions

Version 1.65, released in June 2006, is the latest full release of Oolite. However, the test releases are faster, prettier, more stable and use less memory, so ignoring the full release and using the test releases instead is recommended. Yes, we realise this sounds silly.

↓ Mac OS X

  • Oolite Test Release 1.74
    for Mac OS X 10.4 or later (22 MiB)
  • Oolite Test Release 1.74 updater
    for Oolite 1.65 or test relase 1.70 or later (6.2 MiB)
  • Oolite 1.65
    for Mac OS X 10.3.9 or later (20 MiB)

↓ Windows

  • Oolite Test Release 1.74
    for Windows XP service pack 3 or later (22 MiB)
  • Oolite Test Release 1.74 updater
    for Oolite 1.73.4 only (2.6 MiB)
  • Oolite 1.65
    for Windows XP service pack 3 or later (19 MiB)

↓ Linux

  • Oolite Test Release 1.74
    for 32-bit x86 Linux systems (22 MiB, uses Autopackage)
  • Oolite Test Release 1.74
    for 64-bit x86 Linux systems (22 MiB, uses Autopackage)
  • Oolite 1.65
    for all x86 Linux systems (21 MiB, uses Autopackage)
  • Oolite Test Release 1.74 .deb packages
    specific to Debian 5.04 and Ubuntu 8.04, 9.04 and 9.10.

Source Code and Updates

Source code for all platforms is available from BerliOS. New test releases are announced on the forum.
