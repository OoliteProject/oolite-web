Versions

The latest version of Oolite is beta test release 1.75.2. In case you’re allergic to beta tests, the previous “stable” version, 1.65, is also available. However, 1.75.2 is less crash-prone, more efficicent and supports far more expansion packs, so it is strongly recommended.

↓ Mac OS X

  • Oolite beta 1.75.2
    for Mac OS X 10.4 or later (25 MiB)
  • Oolite 1.65
    for Mac OS X 10.3.9 or later (20 MiB)

↓ Windows

  • Oolite beta 1.75.2
    for Windows XP service pack 3 or later (23 MiB)
  • Oolite beta 1.75.2 updater
    for Oolite 1.75.1 only (1.7 MiB)
  • Oolite 1.65
    for Windows XP service pack 3 or later (20 MiB)

↓ Linux

  • Oolite beta 1.75.2
    for 32-bit x86 Linux systems (24 MiB)
  • Oolite beta 1.75.2
    for 64-bit x86 Linux systems (24 MiB)
  • Oolite 1.65
    for all x86 Linux systems (21 MiB, uses Autopackage)

Source Code and Updates

Source code for all platforms is available from BerliOS. New test releases are announced on the forum.
