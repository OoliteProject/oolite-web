    Tionisla Orbital Graveyard - Shipwrecks
---------------------------------------
This OXP adds some "rusty" ships to the Tionisla Orbital Graveyard. The ship models are based on the newer ship models introduced in Oolite 1.80, designed by Griff, and so will fit in well with those models.

The previous shipwreck pack for the TOGY only included models for the Adder, Asp, Cobra Mk1, Cobra Mk3, Gecko, Krait, Mamba and Moray. This pack includes those ships as well as the Anaconda, Boa, Boa Class Cruiser, Fer-de-Lance, Python and Sidewinder.

In addition to the "rusty" versions, this pack introduces the chance that there might be a new monument made from the core ships.
