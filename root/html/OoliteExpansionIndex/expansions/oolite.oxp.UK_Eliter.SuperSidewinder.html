    DESCRIPTION
------------------

This OXP maked available, as a player and a non-player ship, a considerably souped-up version of Oolite's Sidewinder ship. If InterstellarTweak OXP is installed, then it will make use of the SuperSidewinder.


ACKNOWLEDGEMENTS
-----------------------------

This OXP makes use the inbuilt Sidewinder stuff in Oolite, and borrows a little bit (including the license below) from the 'More Core Player Ships' OXP by Ryan Hoots. Thanks also to Rustem for the exhaust, and for spotting a bug or two.
