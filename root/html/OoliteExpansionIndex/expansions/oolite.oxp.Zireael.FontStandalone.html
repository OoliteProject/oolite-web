readme.txt

The mod takes OCR-A Extended, which is available as open-source from http://sourceforge.net/projects/ocr-a-font/files/ and implements it in the Oolite.

The font was first adapted for Oolite in Deepspace Ships, but seeing as the font is openly available I did not need to use any part of Deepspace OXP itself.

The mod is released under CC-BY 3.0 should anyone want to tweak the font.
