    VimanaShipOverrrrides OXP
-------------
v2.06, by Gnievmir

Makes Vimana HUD the default HUD of many core game ships.

This was part of VimanaHUD OXP in version 2.05, but VimanaHUD version v2.06 is compatible with HUD Selector, adding an alternative way for the player to install VimanaHUD in the player's ship.

Since other ship's properties beyond the HUD are modified, like speed, energy, cargo space and number of pylons, which not all players that might want to use VimanaHUD might wish, those overrides were moved into a separate OXP, letting the player install them if they are welclcome.
