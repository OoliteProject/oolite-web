Planetary Systems Texture Pack B - Cold Deserts
---------------------

This is package of textures for Planetary Systems OXP.
Original textures were reprocessed by Stranger. Reprocessing includes converting JPG file format onto PNG, rescaling image size to standard aspect for lat-long maps 2:1 with size 2 * 2^n width and 2^n height, using color saturation and lights/shadows level correction filters.

Credits:

All original textures in this pack were taken from freebitmaps.blogspot.com. Author of original textures - Robert Stein III.
These textures (rescaled to standard aspect ratio 2:1, but not with 2 * 2^n width and 2^n height) also used in Additional Textures SR: free bitmaps icy|rocky|dusty (author Redspear, spara et al). So I'll list my texture files, original files and Additional Textures SR.
