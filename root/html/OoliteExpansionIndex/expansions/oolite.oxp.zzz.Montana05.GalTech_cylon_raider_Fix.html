Overview
========

Shipbuilders Cylon Raider is loosely based on the primary fighter attack craft of the original series of Battlestar Galactica.
Further information's on the original ship could be found at https://en.battlestarwikiclone.org/wiki/Raider_(TOS)

This package is fixing some bugs, upgrades to features from later versions of Oolite and, inspired by Shipbuilders original
message "Whilst you are free (and encouraged) to re-use any of the scripting, models or texturing in this OXP", includes some
new features, all of them to be removed if Shipbuilder disagrees.

Fixes
=====

included the missing "exhaust", following the Battlestar Galactica design (shipdata.plist)

added some flashers to support engine emission (shipdata.plist)

added shiplibrary.plist to replace the obsolete demoships.plist

some small modifications at the model

Upgrades
========

changed lasers to "multiply" to be similar to the Battlestar Galactica concept (shipdata.plist)

included a normal map, a specular map and modified the diffuse map slightly to work better with the normal map (and to fix some problems)

included "gloss"

New Features
============

As mentioned above this are not part of the original OXP and will be removed if Shipbuilder disagrees:

added Cylon Raider Mark II as squad leader:
    3x fore laser mounts, 1 energy bank more, a bit larger, and a new diffuse and emission map (thanks to cbr)
