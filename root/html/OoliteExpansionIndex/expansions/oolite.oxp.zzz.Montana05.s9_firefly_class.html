Overview
========

This multi-purpose, medium-sized transport ship is a homage to the Firefly Class ship "Serenity" 
( https://firefly.fandom.com/wiki/Firefly_Class ) from the firefly TV series ( https://en.wikipedia.org/wiki/Firefly_(TV_series) ). 
For better adaption to the Oolite universe some modifications took place:

larger hull (88 x 40 x 146), original 52 x 24 x 82
less cargo space (55 t standard + 15 t extension), original 85 t
4 laser mounts (3x fore, 1x rear), original none
2 pylons, original none

Credits
=======

The "Firefly Class Transport" is a tweak of the Firefly test ship from Staer9 (V 0.50 / 2012) so all credits belong to him.

Staer9 (concept, model, most textures)
andrax_iON (view positions)

Remarks
=======

This ship acts as a trader or (sometimes) as a pirate. It is using multiple lasers with 3 laser mounts fore,
the maximum for NPC's are beam lasers.
