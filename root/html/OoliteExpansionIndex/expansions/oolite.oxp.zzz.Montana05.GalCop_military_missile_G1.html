Readme.txt

This is the Military Missile, an enhanced hardened missile.
Faster, more accurate, and costlier than the "hardhead", when ECM'd the mil missile seeks out priority targets, namely Thargoid warships and known pirate vessels.
If it fails to find priority targets in the vicinity, then it defaults to standard hardhead behavior, self-destructing or chasing down it's original target.

Countermeasures: ECM, AMS, fleeing on injectors.

TL 10+
370.0 Cr

For Oolite 1.65 - 1.72+
MilMissile 1.02 - By Wyvern
Reduced speed from 1000 to 800: Improves chances of AMS intercept.
Fixed AI, adds default hardhead mode if priority target mode fails.
