Description Useful packages for new pilots collected by Norby to get the most wanted addons easily. Press \"i\" for the installation list. This pack does nothing after installed once so you can remove it then remove any unwanted packages. Install: Ambience Collection (about 50 graphical and polishment packages), AutoRefuel, Combat Simulator, ETT Homing Beacon, Miner Cobra, Neo-Docklights, Lave Academy, Tionisla Reporter, Traffic Control, Reverse Control, Ring Racer, Safety Catch, Start Advice, The Collector, Welcome Mat. Optional: ILS Instrument Landing System, Market Inquirer, Torus To Sun, Total Patrol, Variable Masslock. Warning: do not install this collection if you plan to read through the whole addon list and can spend enough time to build up your very unique Ooniverse.
