---------------------
Planetary Systems Texture Pack E - Cold Mars
---------------------

This is package of textures for Planetary Systems OXP.
Original textures were reprocessed by Stranger. Reprocessing includes converting JPG file format onto PNG, rescaling image size to standard aspect for lat-long maps 2:1 with size 2 * 2^n width and 2^n height, using of color saturation and lights/shadows level correction filters.
In some cases images were assembled from segments.

Credits:

All original textures in this pack were taken from The Celestia Motherlode (celestiamotherlode.net) - vast repository of Celestia addons (see celestiamotherlode.net/legal.html for legal status of original files). Most of textures based on reprocessed NASA public available images of Solar System planets and moons.
