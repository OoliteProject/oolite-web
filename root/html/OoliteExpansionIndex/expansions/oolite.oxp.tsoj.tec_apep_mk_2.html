tec_apep_mk_2.oxp
-----------------------------------------

An oxp for Oolite v1.80 and higher that adds a new ship called 'TEC Apep Mk II'.

Installing and Uninstalling this oxp
------------------------------------
To install, simply copy the tec_apep_mk_2.oxp folder into your Oolite AddOns folder, next time you start the game the ship will be available.
to uninstall, move or delete the tec_apep_mk_2.oxp folder from your Oolite AddOns folder, next time you start the game the ship will be gone.
