secretarybird.oxp
-----------------------------------------

An oxp for Oolite v1.84 and higher that adds a new ship called 'Secretarybird'.

Installing and Uninstalling this oxp
------------------------------------
To install, simply copy the secretarybird.oxp folder into your oolite AddOns folder, next time you start the game the ship will be available.
to uninstall, move or delete the secretarybird.oxp folder from your oolite AddOns folder, next time you start the game the ship will be gone.
