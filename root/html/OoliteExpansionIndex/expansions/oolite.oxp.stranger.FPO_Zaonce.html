Famous Planets Overhaul: Zaonce

This OXP changes visual appearance of Zaonce using custom 4096 x 2048 textures for diffuse and specular maps.
Zaonce has more rough terrain comparing with Lave, so terrain shadows are quite noticeable. Moreover, original terrain normal map already has exaggerated magnitude to provide more visual effect.

Conflicts and dependencies

Any packages using script to redefine Zaonce texture, such as Povray Planets, will overwrite FPO Zaonce texture.
In case of custom Zaonce textures redefined in planetinfo.plist result will be dependent from loading order. The latest planetinfo.plist in loading order has highest priority.
In any case having one custom texture for one planet, not several, is right.

Credits

Textures for diffuse map from NASA Blue Marble open project.
