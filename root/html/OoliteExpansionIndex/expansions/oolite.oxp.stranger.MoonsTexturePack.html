---------------------
Moons Texture Pack
---------------------

This is package of textures for Moons OXP.
Original textures were reprocessed by Stranger. Reprocessing includes converting JPG file format onto PNG, rescaling image size to standard aspect for lat-long maps 2:1 with size 2 * 2^n width and 2^n height, using of color saturation and lights/shadows level correction filters.

All original textures were taken from The Celestia Motherlode (celestiamotherlode.net) - vast repository of Celestia addons (see celestiamotherlode.net/legal.html for legal status of original files). These textures based on reprocessed NASA public available images of Solar System moons. There is problematic to refer exactly author of every texture sample, so I'll just refer to authors of relevant addons.
