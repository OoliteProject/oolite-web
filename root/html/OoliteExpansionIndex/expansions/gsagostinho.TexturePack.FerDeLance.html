gsagostinho's Texture Pack: Fer-de-Lance
========================================

Version: 1.5
Required Oolite Version: 1.80
Author: Gilberto Agostinho (gsagostinho)
Credits: based on Griff's Fer-de-Lance.
License: CC-BY-NC-SA 4.0

About
=====

This bundle adds 12 new paint jobs for the Fer-de-Lance for both the player and the NPCs. This work is based on Griff's Fer-de-Lance.

Version history
===============

14/06/2019 - Version 1.5: three new paint jobs.
08/10/2018 - Version 1.4: better reflection settings (thanks another_commander!)
23/07/2018 - Version 1.3: improvements to the original liveries.
                          adjustments to specular_color and shininess.
21/07/2018 - Version 1.2: three new liveries.
19/07/2017 - Version 1.1: cockpit windows less yellow.
                          removed unused texture.
18/07/2017 - Version 1.0: initial release.
