Equipment Mining IFF Scanner Upgrade

Show Asteroids, Boulders and Splinters in alternative scanner colours for easier tracking when mining. Asteroids are white/gray, boulders white/orange and splinters white/green (safe to pick up ). For an additional fee, we can hack the code to this scanner upgrade to distinguish cargo canisters and escape pods also.
