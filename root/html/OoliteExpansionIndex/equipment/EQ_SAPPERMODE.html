Equipment Sapper Mode

This primable equipment can switch your Sniper Gun between Sniper and Sapper mode when activated. You get this equipment with Sapper weapons for free, but you must fix this if Sapper Mode get damage else you can use Sniper mode only. The mode key can set auto, preset, target distance, heat or manual Sapper modes.
